Setup

One-time setup (takes ~2 minutes):
1. In Entra → App registrations, click New registration. Any name works.
2. Supported account types: Accounts in this organizational directory only.
3. Redirect URI: pick Single-page application (SPA) and paste the "Redirect URI" value above (defaults to this page's URL — host this HTML somewhere, or use http://localhost:port/… and serve it via a tiny local server).
4. After creation, go to API permissions → Add a permission → Microsoft Graph → Delegated → add Directory.Read.All. Click Grant admin consent.
5. Copy the Application (client) ID from the Overview page into the field above. Sign in — Global Reader on your account is sufficient at runtime.
